# AutoDeleteFileOnPCWechat

自动删除 PC 端微信自动下载的大量文件、视频、图片等数据内容，解放你的空间。

删除过程中如出现转圈等现象请耐心等待，删除结束将会有提示。

## 特性
1. 自由设置想要删除的文件类型，包括图片类缓存、文件、图片、视频；
2. 自由设置需要删除的文件的距离时间，默认 24 个月；
3. 删除后的文件放置在回收站中，检查后自行清空，防止删错需要的文件；
